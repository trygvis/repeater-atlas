# Page Specs

Design specs for individual pages. Each document describes the purpose, routes,
layout, data sources, and behavior for one page or page group.

- [map.md](map.md) — Primary public map view (`/`); repeater markers, left/right
  panes, search modal, login link, and viewport persistence.
- [my-page.md](my-page.md) — Authenticated user account page (`/-/my`); saved
  QTH locations, CHIRP export, and portable logbook PDF generator.
- [repeater-detail.md](repeater-detail.md) — Public repeater detail page
  (`/$CALL_SIGN`); services grouped by kind, linked repeater network map, and
  nearby repeaters within 50 km.
