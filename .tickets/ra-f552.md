# New feature: Personal logbook

From "my page", add a section to download your own personal logbook.

Optional features should be configured before downloading the logbook.

## Requirements

- The page size should be customizable, A4 or A5.
- Landscape orientation
- Page header: A field to write the current date and location
- Page footer: Print the user's call sign and an field to write page number
- These are the main fields for the table:
  - QSO #
  - Start
  - End
  - Station
  - Operator
  - QTH
  - RST sent (a narrow field, for 3 digits)
  - RST received, same as RST sent
  - Freq
  - Mode
  - Power
  - Remarks
- Number of log pages, default to 10
- Optional first page: Title page
  - Print "Logbook for $CALL"
- Optional last page: List of your own locations
  - Each location contains address, maidenhead and lat/long

## Non-functional requirements

- The logbook should be generated with Typst, a tool to generate PDFs from a
  text format.
- The `typst` command line interface can be expected to be available on the
  command line.
- The PDF should be generated on the fly.
