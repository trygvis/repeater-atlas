# Visualize linked repeaters

## Club page

On the club page, there shall be a map showing all repeaters.

Also, all links between repeaters (and their non-club repeaters) should be shown
as well. The non-club repeaters should have a red-ish color instead of the
standard blue color.

## Repeater page

If a repeater is linked, make a separate map that shows all the repeaters in the
network, with lines between the linked repeaters. The map should be show all
repeaters in the network.
